关于谷歌 Chrome 浏览器碰见的一些小问题

2025-12-05 2 min read

前段时间一直在用edge，结果因为edge的一些不太好的东西（游戏助手）导致转了Firefox，可是Firefox的使用体验上又不太好，最后转了Chrome。最近在用Chrome的时候碰见了点之前版本没有的小问题（版本142），如下：

  1. 点击正在播放的标签页的喇叭图标，标签页不静音
  2. 浏览器无法检测到电脑中正在运行的QQ客户端进行快速登录

解决方法如下：

  • 点击正在播放音频的标签页的喇叭图标时，标签页无法静音
    • 解决方法：在地址栏输入chrome://flags/，在搜索框中输入tab audio muting，找到Tab Audio Muting UI Control项，将其设置为Enabled，重启浏览器即可。
  • 浏览器无法使用QQ快捷登录（检测不到电脑中已启动的QQ客户端）
    • 解决方法：在地址栏输入chrome://flags/，在搜索框中输入local network access checks，找到Local Network Access Checks项，将其设置为Enabled ，重启浏览器即可。

至少我目前发现这两个问题，第一个不知道什么时候更新的，但是第二条据查是在142版本中更新了本地网络的访问限制，具体请见：Chrome版本142 | 版本日志
